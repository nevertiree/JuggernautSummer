##1.新建MyBatisGenerator项目

1.新建IDEA项目

使用Maven新建，并且输入Maven版本号

```xml

<dependency>
    <groupId>org.mybatis.generator</groupId>
    <artifactId>mybatis-generator</artifactId>
    <version>1.3.3</version>
</dependency>

```
